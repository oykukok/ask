# Contributing to ask

Thanks for your interest in contributing!

## How to contribute

1. Fork the repository
2. Create a new branch for your change (`git checkout -b my-change`)
3. Make your changes
4. Commit with a clear message (`git commit -m "describe what you did"`)
5. Push to your fork and open a pull request

## Guidelines

- Keep the script simple and dependency-free (only curl and jq)
- Test your changes before submitting
- Update the README if your change affects how the tool is used
- Write clear commit messages

## Reporting bugs

Open an issue on GitHub describing the problem and how to reproduce it.
